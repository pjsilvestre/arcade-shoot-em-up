# arcade-shoot-em-up

## Latest Release (Windows)

[1.0.0](https://github.com/pjsilvestre/arcade-shoot-em-up/releases/tag/1.0.0)

To play:

* Download `arcade-shoot-em-up-1.0.0.zip`
* Unzip `arcade-shoot-em-up-1.0.0.zip`
* Run `arcade-shoot-em-up.exe`

## Controls

* Movement: WASD
* Shooting: Space Bar
